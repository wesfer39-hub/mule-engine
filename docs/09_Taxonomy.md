# 🗂 Mule Engine – Collapse Taxonomy  
### 붕괴 유형 분류표

---

## Ⅰ. 목적

이 문서는 Mule Engine 아카이브에 축적된 사례들을  
👉 **“붕괴 구조 유형”으로 분류하기 위한 기준표**다.

이 분류는 승패나 시대가 아니라  
**“어디서 무엇이 먼저 무너졌는가”**를 기준으로 한다.

---

## Ⅱ. 기본 붕괴 축 (Primary Axes)

---

# 🔴 T1 — Survival Collapse  
### (L2: 군수 / 생존 붕괴)

**정의:**  
유지 구조가 먼저 붕괴하여 군대가 생존 집단으로 전환

**핵심 신호**
- NetGain ≤ 0 지속
- 보급 차단 / 고갈
- Recovery Window 소멸

**전형 사례**
- 크라수스 파르티아
- 스탈린그라드 포위
- 나폴레옹 러시아 원정(후반)

---

# 🟠 T2 — Biological Shock Collapse  
### (L3: 생물 / 물리 붕괴)

**정의:**  
환경·피로·손상으로 기능이 먼저 붕괴

**핵심 신호**
- 급격한 체온/탈수/피로
- 이동·전투 능력 급락
- 회복률 붕괴

**전형 사례**
- 트레비아
- 혹한/사막 쇼크 전투

---

# 🟡 T3 — Perception / Control Collapse  
### (L4: 지각 / 통제 붕괴)

**정의:**  
군대가 “하나의 군대”로 반응하지 못하게 되는 붕괴

**핵심 신호**
- 시야/정보 붕괴
- 명령 전달 실패
- 부대 분절

**전형 사례**
- 트라시메네
- 테우토부르크 숲

---

# 🟢 T4 — Operational / Mass Collapse  
### (L5: 운영 / 질량 붕괴)

**정의:**  
군대가 통제 시스템이 아닌 “질량”으로 전환

**핵심 신호**
- 병력 과밀
- 재배치 불능
- 처리량 붕괴
- 병목 자기증폭

**전형 사례**
- 칸나이
- 대규모 포위 섬멸

---

## Ⅲ. 복합 붕괴 유형 (Hybrid Types)

---

# 🔴🟠 H1 — Chronic Survival–Biological Collapse  
(L2 + L3)

**특성**
- 원정 전체가 임계 상태
- 지속 손실
- 회복 창 반복 개폐

**전형 사례**
- 제1차 십자군 소아시아 횡단
- 나폴레옹 러시아 원정(초중반)

---

# 🟠🟡 H2 — Shock–Disorientation Collapse  
(L3 + L4)

**특성**
- 환경 쇼크 → 통제 붕괴
- 짧은 시간 내 전면 붕괴

**전형 사례**
- 혹한/안개 매복 사례

---

# 🟡🟢 H3 — Control–Mass Collapse  
(L4 + L5)

**특성**
- 통제 상실 → 병목 고착 → 질량화

**전형 사례**
- 테우토부르크 숲
- 도시 포위전 초기 붕괴

---

# 🔴🟢 H4 — Isolation–Saturation Collapse  
(L2 + L5)

**특성**
- 고립 + 처리량 부족
- 구조적 섬멸

**전형 사례**
- 스탈린그라드
- 일부 포위 섬멸전

---

## Ⅳ. 아카이브 표기 규칙

각 Case에는 다음을 명시한다:

- Primary Type: T?
- Hybrid Type (있다면): H?
- Collapse Sequence
- Locked Collapse 여부

예:

Primary: T3  
Hybrid: H3  
Sequence: L4 → L5 → L3  
Locked: True

---

## Ⅴ. 장기 목표

이 분류표는 사례가 쌓일수록:

- 붕괴 유형 지도
- 원정 붕괴 계통도
- 실패 구조 진화도

로 발전한다.

> **전쟁이 속한 ‘붕괴 종’(species)을 만드는 작업**이다.
